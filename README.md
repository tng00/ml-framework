# Лабораторные работы по предмету Фреймворки ИИ
Выполнил: Нгуен Ньы Хоанг Ань (группа М8О-409Б-22).
В этом репозитории собраны 5 лабораторных работ, посвященных исследованию и сравнению различных алгоритмов машинного обучения на одних и тех же данных.

## Цель
Пройти полный путь от простых метрических алгоритмов до градиентного бустинга, реализовать каждый алгоритм "с нуля" и сравнить их эффективность с библиотечными решениями (Scikit-learn).

## Структура репозитория

Все работы выполнены на двух датасетах:
1.  **Классификация:** [Banknote Authentication](https://archive.ics.uci.edu/ml/datasets/banknote+authentication) (определение подлинности купюр).
2.  **Регрессия:** [Medical Cost Personal Datasets](https://www.kaggle.com/datasets/mirichoi0218/insurance) (предсказание медицинских расходов).
